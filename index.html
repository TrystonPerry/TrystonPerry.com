Github User Search

Single page Bootstrap 4 site that uses Javascript ES6 & ES7 Syntax to fetch Github User Info asynchronously. Just type a valid username in the search box and an event listener will call a fetch request to the Github API with that username as a parameter. Updates code instantaneously displaying user info and any corresponding repositories dynamically.

See Full Site View Repository
CryptoMarketPlace

WebApp that uses the Request.js to get top 100 cryptocurrency price information from API and display it in a bootstrap 4 styled table. Uses numbro.js to shorten numbers and modify number appearances based on value. Gets corresponding coin logo based on coin symbol from API. Create or login into an account to favorite certain cryptocurrencies using an AJAX Requests to send data without reloading the page.

Yelp for Campgrounds

YelpCamp is a website that implements RESTful Routing techniques. Features user account creation, login, get, post, update, and delete requests. Uses Bootstrap 3 for design, and stores data in a MongoDB non-relational database. All user-provided data is cleansed of any harmful data. And User passwords are never stored in plaintext, instead, are hashed using Passport.js.
